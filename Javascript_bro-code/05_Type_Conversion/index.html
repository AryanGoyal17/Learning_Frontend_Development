Type Conversion


        Type conversion = change the datatype of a value to another(strings, booleans, numbers)


        Important: When we accept user input in Js, it gets stored as a string by default(like in Python)


        Js-code -- 1:
        User input is taken as string by default and on adding 1, string concatenation Occurs


        Js-code -- 2:
        User input got taken as string by default but since ++(increment) is an operation on numbers.. the age variable got forcibly converted to number data type and incremented.


        Js-code -- 3:

        NaN -- Not a Number(when string converted to number)
        If the string is not empty then on converting to boolean it returns True!


        Js-code -- 5:

        Empty string when type-casted to Number gives 0

        Type-casting can be used to check whether the string entered by user is empty or not..
        If the string is empty, then on type-casting to boolean it returns False otherwise True!!
