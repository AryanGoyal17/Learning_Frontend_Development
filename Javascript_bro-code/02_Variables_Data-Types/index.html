Variables and Data types in Js 


        Variable -- A container that stores a value
                 -- Behaves as if it were the value it contains

        (A)First we create a variable, and after that we assign a value to it 

        Data Types:
        1-- Number(integer, decimal etc..)
        2-- String(can contain numbers but the numbers in a string cant be used for arithmetic calculations)
        3-- Boolean(True/False) -- generally used as flags or in if-else statements instead of direct output

        For Printing the data type of a variable use typeof keyword
