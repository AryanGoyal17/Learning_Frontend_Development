Arithmetic Operators in JavaScript

        Arithmetic Operators:

        1--Operands(values, variables etc..)
        2--Operators(+, -, *, / etc..)

        


        Different arithmetic operators--

        +, -, *, /, **(exponent), %(modulus)

        


        Augmented assignment operators -- +=, -=, *=, /=, **=, %=

        Instead of writing students = students * 5... we can do students *= 5 ... 

        


        The original value of variable gets updated(changed) on applying arithmetic ops..

        


        ++ is increment operator and -- is decrement operator

        


        Operator Precedence:

        1. Paranthesis
        2. exponent(**)
        3. *, /, %
        4. +, -
